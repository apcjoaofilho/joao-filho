# Como eu uso Obsidian como centro de operação

Eu não uso Obsidian só para guardar texto.

Eu uso como centro de operação.

É onde projeto nasce, ganha contexto, acumula decisões, recebe links úteis, vira plano e depois se conecta com execução.

## O que eu quero evitar

O problema que eu tento resolver é simples:

- ideia solta em um lugar
- tarefa em outro
- referência em outro
- contexto na cabeça

Isso destrói continuidade.

Quando você volta para um projeto alguns dias depois, gasta energia demais só para lembrar onde estava.

## O papel do Obsidian no meu sistema

Hoje eu uso Obsidian para quatro funções principais:

1. organizar contexto
2. registrar decisões
3. estruturar entregáveis
4. alimentar publicação e comunicação

Isso vale tanto para projeto técnico quanto para conteúdo.

## O que faz diferença na prática

### 1. MOCs em vez de depender só de pastas

Pasta ajuda a armazenar.
MOC ajuda a pensar.

Quando o projeto cresce, eu quero uma nota que funcione como painel:

- o que existe
- o que importa
- o que vem depois

Isso reduz muito o custo de reentrada.

### 2. Notas operacionais, não só conceituais

Uma nota boa para trabalho não precisa ser bonita.
Ela precisa ser útil.

Por isso eu prefiro notas que respondem perguntas como:

- qual é o objetivo
- qual é o estado atual
- o que já foi testado
- onde estão os links críticos
- qual foi a última decisão tomada

### 3. Obsidian como memória de projeto

Quando estou lidando com vários projetos ao mesmo tempo, o vault funciona como memória externa.

Isso é especialmente útil quando o trabalho envolve:

- agentes
- automações
- deploys
- conteúdo
- documentação

Sem isso, o contexto evapora rápido.

### 4. Conteúdo nasce melhor quando nasce de nota

Eu gosto mais de produzir conteúdo a partir de notas do que começar direto na rede social.

A nota permite:

- pensar sem pressão de formato
- aprofundar argumento
- guardar exemplos e links
- reaproveitar depois

Depois eu transformo a nota em post, carrossel, thread ou roteiro.

## Por que isso funciona para mim

Porque Obsidian me deixa juntar três camadas no mesmo lugar:

- pensamento
- operação
- publicação

Essa combinação é poderosa.

Em vez de tratar conhecimento, execução e distribuição como mundos separados, eu tento conectar os três.
